## Working across machines

This repo is developed on more than one machine, and **git is the only thing that syncs.** Claude's auto-memory (`~/.claude/`), the app's local SQLite DBs, and `.venv/` do NOT travel between machines â€” never treat memory as shared truth (that's how the "grading is dev-only" note went stale).
