**NEVER add a Co-Authored-By tag, Signed-off-by, or any line that references Claude or an Anthropic email address to any git commit.**

## Working across machines

This repo is developed on more than one machine, and **git is the only thing that syncs.** Claude's auto-memory (`~/.claude/`), the app's local SQLite DBs, and `.venv/` do NOT travel between machines ŌĆö never treat memory as shared truth (that's how the "grading is dev-only" note went stale).

- **End of session:** commit + push everything. If a feature's status or gating changed, update this file in the **same commit**.
- Full checklist + what does/doesn't travel: `docs/dev_notes.md` ŌåÆ "Cross-machine workflow".
